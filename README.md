# Create a Local Server with Node.js

A simple HTTP server built using Node.js core modules. This project demonstrates the fundamentals of backend development, including routing, handling requests and responses, setting HTTP headers, and serving HTML content without using external frameworks.

## 🚀 Features

* Create a local HTTP server using Node.js
* Handle multiple routes (`/` and `/about`)
* Serve HTML content to the browser
* Custom 404 page for invalid routes
* Proper HTTP status codes and headers
* Lightweight implementation without Express.js

## 🛠️ Technologies Used

* Node.js
* JavaScript
* HTTP Module

```bash

```

3. Start the server:

```bash
node server.js
```

4. Open your browser and visit:

```bash
http://localhost:3000
```
